# ncdu

今天被别人安利一个在终端查看、清理内存小工具 `ncdu` ，发现炒鸡好用。

### 安装

打开终端，输入： `brew install ncdu`

### 使用：

安装好以后，直接在终端使用命令 `ncdu .` 就可以了， 可以很方便查看内存情况。

![](https://tva1.sinaimg.cn/large/006y8mN6ly1g74ups3n0gj30rw0h6gsz.jpg)



###  常用方法

- enter: 打开选定目录
- n : 按文件名排序（升序/降序）
- s: 按文件大小排序（升序/降序）
- C: 按项目数排序（升序/降序）
- d: 删除选定的文件或目录
- g: 以图形方式显示百分比
- q: 退出（也可以用control+z）

如果想查看更多方法 `ncdu -h`
